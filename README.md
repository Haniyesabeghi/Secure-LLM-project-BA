# [loading Maral LLM And Dorna LLM]

Easily and quickly load and use native Persian models for Large Language Model (LLM) tasks.

## Project Description

This repository contains tools and scripts that facilitate loading and utilizing Persian language models. Native Persian models can include pre-trained models for tasks such as text classification, named entity recognition (NER), machine translation, summarization, and more. The goal of this project is to simplify the process of using these models for Persian-speaking developers and researchers.

## Features

- Support for Persian language models (e.g., Hugging Face models, custom models, etc.)
- Ready-to-use code samples for common LLM tasks
- Usable in various environments (VSCode, Colab)

## Installation

To install and set up, first clone the repository.
